# GitHub App 作成手順

このツールで必要なGitHub Appの作成手順の概略です。

## 1. 作成画面を開く
- GitHub: `Settings > Developer settings > GitHub Apps > Create GitHub App`

## 2. 基本情報を入力
- **GitHub App name**: 例 `Custom org role permissions fetch`
- **Description**: アプリの用途を記載
- **Homepage URL**: アプリのホームページURL

## 5. Permissions を設定
- **Organization permissions**
  - `Custom organization roles` = `Read-only`
